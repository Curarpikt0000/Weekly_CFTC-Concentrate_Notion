每周执行

CFTC（美国商品期货交易委员会）持仓报告 的自动化采集与归档工具。它与前一个绘图脚本不同，这个脚本更侧重于合规性数据备份和大户筹码追踪。

一、 这段代码的主要实现功能
代码通过访问 CFTC 官网，抓取“金属与其他商品”的期货与期权合并报告（COT Report），具体逻辑如下：

自动化抓取 (Data Scraper)： 实时访问 CFTC 官网最硬核的文本报告页面（other_lof.htm）。

时间精准对齐 (Date Extraction)： 脚本会从混乱的文本中利用正则表达式（Regex）精准提取该报告的实际发布日期。

注意：CFTC 报告有滞后性，通常在周五发布周二的数据，脚本确保记录的是数据日期而非下载日期。

云端归档 (GitHub Persistence)： 将抓取的原始文本保存为 .txt 文件并推送到你的 GitHub 仓库。

Notion 数据库同步： 在 Notion 中创建一个条目，关联 GitHub 的原始文件链接。
